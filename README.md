# Sample-Store

2026/07/17

HTML・SCSS・JavaScriptで制作した架空のコンビニサイトを、WordPressのオリジナルテーマとして構築した作品です。

## デモ

https://nmhr12.com/sample-store/

## 使用技術

- WordPress
- PHP
- HTML
- SCSS
- JavaScript
- Contact Form 7

## 実装内容

- オリジナルテーマ開発
- レスポンシブ対応
- ハンバーガーメニュー
- お問い合わせフォーム
- カスタム投稿（News）
- カテゴリー一覧・カテゴリー別表示
- 記事詳細ページ
- 一覧ページ
- ページネーション
- 管理画面から更新できる構成

## 工夫した点

静的サイトをWordPressへ移行し、更新しやすいサイト構成を意識しました。
共通パーツをテンプレート化し、ニュースはカスタム投稿で管理できるようにしています。
カテゴリー機能を実装し、目的の記事を探しやすい設計にしました。

## 今後追加したい機能

- カスタムフィールドの実装
- SEOを意識した構造改善
- パフォーマンス改善
- セキュリティ対策の強化

## 制作目的

WordPressオリジナルテーマ開発の学習を目的として制作しました。
静的コーディングだけでなく、CMS化を通して更新性や保守性も意識して実装しています。
